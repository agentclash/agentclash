# @agentclash/cli-__TRIPLE__

Prebuilt AgentClash CLI binary for `__OS__-__CPU__`.

This package is not meant to be installed directly. It ships as an optional
dependency of the [`agentclash`](https://www.npmjs.com/package/agentclash)
package; npm's `os`/`cpu` filters install only the binary that matches the
host.

Install the CLI via:

```bash
npm i -g agentclash
```

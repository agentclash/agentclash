Thought for 5s
In guitar tab, 2h3p2 means:
Play fret 2 → hammer-on to fret 3 → pull-off back to fret 2

So your line:

2 2 2 0 2h3p2

means:

Play 2nd fret
Play 2nd fret
Play 2nd fret
Play open string (0)
Then play 2nd fret, quickly hammer-on to 3rd fret, then pull-off back to 2nd fret
Basically that last part is a tiny ornament/flick:
2 → 3 → 2
but you only pick the first 2. The 3 and return to 2 are done with your fretting hand.

So if it’s on one string, it sounds like:

pick 2, tap 3, release back to 2

Very Bowie-ish little flourish, honestly. Finger-wise, usually:

Index finger on 2nd fret
Middle finger hammers onto 3rd fret
Then middle finger pulls off so index still holds 2nd fret

Yep, this is guitar tab, not staff notation. Let’s decode it properly.
The visible strings are:

E|  ← 1st string, thinnest
B|  ← 2nd string
G|  ← 3rd string
D|  ← 4th string
Numbers = frets.
So:
G|---7-7-6-6-4-4---
means play the 3rd string, frets:
7, 7, 6, 6, 4, 4

Not all strings at once — only the string where the number appears.

Symbols in this tab

Slide up from fret 5 to fret 7.
Play fret 5, keep pressure, slide finger to fret 7.

4\2
Slide down from fret 4 to fret 2.
6p4
Pull-off from fret 6 to fret 4.
You pick fret 6, then pull your finger off so fret 4 rings. Your finger must already be holding fret 4.

2h3p2
Hammer-on and pull-off.
Pick fret 2, hammer fret 3, pull back to fret 2.

Thought for 7s
Yesss. Let’s make hammer-on and pull-off stupid simple.
Use one string only first. Take the 1st string = thinnest string or 3rd string, whichever feels easier.

1. Hammer-on = h
Example:
2h3
Meaning:
Pick 2nd fret once → hit 3rd fret with another finger without picking again

Finger placement
For 2h3:
Put index finger on 2nd fret
Pick the string
Then slam/press your middle finger onto 3rd fret
So:
2h3 = index on 2 → pick → middle finger hits 3
Important: don’t just gently place the finger. You have to hammer it with enough force so the note rings.
Practice drill
E|--2h3--2h3--2h3--2h3--
Say it like:
pick-hit, pick-hit, pick-hit

Only pick the 2. Do not pick the 3.

2. Pull-off = p
Example:
3p2
Meaning:
Pick 3rd fret → pull finger off so 2nd fret rings

Finger placement
Before playing, hold both fingers:
Index finger already on 2nd fret
Middle finger on 3rd fret
Then:
Pick the string while holding 3rd fret
Pull the middle finger off slightly downward/sideways
The note returns to 2nd fret
So:
3p2 = hold index on 2 + middle on 3 → pick 3 → pull middle off → 2 rings
The biggest mistake: people just lift the finger.
Don’t lift. Pluck the string slightly with the pulling finger as you remove it.
Tiny “flick” motion.

Practice drill
E|--3p2--3p2--3p2--3p2--
Say:
pick-flick, pick-flick, pick-flick

Only pick the 3. Do not pick the 2.

3. Hammer + pull together = 2h3p2
This is what you saw earlier.
2h3p2
Meaning:
Pick 2 → hammer 3 → pull back to 2

Finger setup
Index finger on 2nd fret
Middle finger ready above 3rd fret
Steps:
Hold 2nd fret with index
Pick the string
Hammer middle finger onto 3rd fret
Pull middle finger off so 2nd fret rings again
So the sound is:
2 → 3 → 2
But your picking hand only does:
pick → no pick → no pick
Super slow exercise
Do this for 5 minutes:
E|--2h3p2----2h3p2----2h3p2----
Count it like:
ONE-and-ah   ONE-and-ah
2   3   2    2   3   2
Or even simpler:
pick-hit-flick

That’s the whole technique.

Make it easier first
Start on the 1st string, 2nd fret and 3rd fret:
E|--2h3--
E|--3p2--
E|--2h3p2--
Then try the same on the 3rd string, because your Hindi/Bollywood tabs use the 3rd string a lot:
G|--4h6--
G|--6p4--
G|--4h6p4--
For 4h6p4:
Index on 4th fret
Ring finger hammers/pulls on 6th fret
Why yours may sound dead
Most common issues:
Hammer-on sounds weak:
You are placing the finger too softly. Hit the fret fast and close to the metal fret wire.

Pull-off doesn’t ring:
You are lifting the finger straight up. Pull slightly downward like you’re plucking the string with that finger.

Wrong note rings/muting happens:
Keep the lower finger pressed. For 3p2, your index must stay firmly on 2nd fret.

Buzzing:
Press closer to the fret wire, not in the middle of the fret box.

The one line to remember:
Hammer-on = hit the next fret without picking.
Pull-off = flick off to the lower fret without picking.

Thought for 6s
Yes bro — 2h3p2 = 2h3 hammer + 3p2 pull-off.
Not 2h33p2.

Think of it like:

2 h 3 p 2
Meaning:
Put index on 2nd fret
Pick the string once
Hammer middle finger onto 3rd fret
Pull middle finger off so it returns to 2nd fret
So the sound is:
2 → 3 → 2
But picking hand does only:
pick → no pick → no pick
The p2 means “pull off from the note you’re currently on back to 2.”
Since after 2h3, you are on 3, p2 automatically means 3p2.
So mentally read it as:

2h3p2 = 2h3 + 3p2
Not:
2h3 + 3p2 with extra 3 picked
Tiny mantra: pick-hit-flick.

Thought for 21s
For guitar-learning context: “Seven Nation Army” is the perfect beginner riff because it’s mostly single notes. No hammer-ons, no pull-offs, no / slides needed at first.
Beginner version:

5th string = A string = second thickest string

A|--7--7--10--7--5--3--2--
Play only the 5th string.
Finger placement:

7  = index finger
10 = pinky or ring finger
5  = index finger
3  = index finger
2  = index finger
Rhythm/vibe:
7    7  10  7   5   3   2
DUM  DUM da-DUM DUM DUM DUM
Don’t rush it. The first 7 is held slightly longer, then the second 7-10-7 feels tighter.
Also fun fact: that riff sounds like a bassline, but it was actually played on guitar by Jack White with an octave effect. Absolute beginner-guitar canon lol.

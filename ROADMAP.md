# Roadmap

The authoritative, always-current roadmap lives in a pinned GitHub issue:

➡️ **[Roadmap (issue #157)](https://github.com/agentclash/agentclash/issues/157)**

It's kept in sync automatically — `.github/workflows/roadmap-sync.yml` updates the
roadmap issue as linked issues are closed and reopened.

## How to influence it

- Comment on [#157](https://github.com/agentclash/agentclash/issues/157) or open a
  [Discussion](https://github.com/agentclash/agentclash/discussions) to propose
  direction.
- File a [feature request](https://github.com/agentclash/agentclash/issues/new/choose)
  for something concrete.
- Browse [`good first issue`](https://github.com/agentclash/agentclash/labels/good%20first%20issue)
  to pick up roadmap-adjacent work.

This file is intentionally thin so the roadmap never lives in two places — see #157
for the current plan.

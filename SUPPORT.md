# Support

Thanks for using AgentClash! Here's where to go for help.

- **Questions & ideas** → [GitHub Discussions](https://github.com/agentclash/agentclash/discussions)
  (use the Q&A category). Best for "how do I…", design questions, and show-and-tell.
- **Bugs & feature requests** → [open an issue](https://github.com/agentclash/agentclash/issues/new/choose)
  using the bug or feature form.
- **Security vulnerabilities** → do **not** open a public issue. See [SECURITY.md](SECURITY.md)
  to report privately.
- **Documentation** → https://www.agentclash.dev/docs (quickstart, guides, CLI reference,
  challenge-pack authoring).
- **Want to contribute?** → [CONTRIBUTING.md](CONTRIBUTING.md). New here? Start with a
  [`good first issue`](https://github.com/agentclash/agentclash/labels/good%20first%20issue).

Please be patient — AgentClash is maintained by a small team. Searching existing
issues and discussions first is the fastest way to an answer.
